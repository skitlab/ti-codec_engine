Engine.h File Reference

Detailed Description

The Codec Engine Runtime.

Remarks:
Provides the user an interface to open and manipulate an Engine which can instantiate and communicate with XDAIS algorithms that run either on the local CPU or on a "remote" Server.
#include <ti/sdo/ce/Server.h>
