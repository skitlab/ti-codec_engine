Codec Engine API Reference

ce-t06

Modules

The Codec Engine is partitioned into several components:

  • The Codec Engine Runtime (CERuntime_* and Engine_*) - Provides codec-independent services.
  • VISA - Alg-independent user APIs (VISA_*) - A set of services that can be used on algorithms of any class.
  • Codec Engine Server Interface (Server_*) - Provides an interface to Servers containing remote algorithms.
  • The latest VISA APIs - Providing codec-specific interfaces, reflecting the XDM interfaces for the following classes:
    • Video
      • Encoding
        • VIDENC2 - Video Encoder Interface (preferred)
        • VIDENC1 - Video Encoder Interface (supported)
      • Decoding
        • VIDDEC3 - Video Decoder Interface (preferred)
        • VIDDEC2 - Video Decoder Interface (supported)
    • Imaging
      • IMGDEC1 - Image Decoder Interface
      • IMGENC1 - Image Encoder Interface
    • Speech
      • SPHDEC1 - Speech Decoder Interface
      • SPHENC1 - Speech Encoder Interface
    • Audio
      • AUDDEC1 - Audio Decoder Interface
      • AUDENC1 - Audio Encoder Interface
    • Video Trancoding
      • VIDTRANSCODE - Video Transcoder Interface
  • Older VISA APIs continue to be supported, but to align with the XDM 0.9 roadmap, they are deprecated in this release. These interfaces are comprised of the following:
    • Video
      • VIDDEC1 - Video Decoder Interface
      • VIDDEC - Video Decoder Interface
      • VIDENC - Video Encoder Interface
    • Imaging
      • IMGDEC - Image Decoder Interface
      • IMGENC - Image Encoder Interface
    • Speech
      • SPHDEC - Speech Decoder Interface
      • SPHENC - Speech Encoder Interface
    • Audio
      • AUDDEC - Audio Decoder Interface
      • AUDENC - Audio Encoder Interface
  • Also, this release includes Beta support for the following XDM interfaces:
    • VIDANALYTICS - Video Analytics Interface
    • UNIVERSAL - Universal Algorithm Interface

Additionally, there are several utility modules provided:

  • Memory - OS Independent Memory Interface
