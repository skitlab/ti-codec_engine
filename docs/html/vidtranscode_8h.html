vidtranscode.h File Reference

Detailed Description

The VIDTRANSCODE video transcoder interface. Provides the user an interface to create and interact with XDAIS algorithms that are compliant with the XDM-defined IVIDTRANSCODE video transcoder interface.

#include <ti/xdais/dm/xdm.h>
