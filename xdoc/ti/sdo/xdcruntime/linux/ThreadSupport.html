ARGUMENTS
p1 — priority one
p2 — priority two
eb — Pointer to Error.Block
DETAILS
Refer to xdc.runtime.knl.Thread.compareOsPriorities for more details.
  • CompStatus_ERROR if an error occured
  • CompStatus_LOWER if p1 is lower than p2
  • CompStatus_EQUAL if p1=p2
  • CompStatus_HIGHER if p1 is higher than p2 details.
